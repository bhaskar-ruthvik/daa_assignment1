Convex Hull

The convex-hull problem entails finding the convex-hull of a given set of points.A convex hull is the smallest convex polygon that encloses all the points in the set.

Example Outputs

Test Case 1:

KirkPatrick-Seidel:

Input: (0,2) (4,4) (4,0) (6,2) (2,4) (2,0)

Output: (0,2) (2,4) (4,4) (6,2) (4,0) (2,0)

Jarvis March:

Input: (0,2) (4,4) (4,0) (6,2) (2,4) (2,0)

Output: (4,0) (6,2) (4,4) (2,4) (0,2) (2,0)

Test Case 2:

KirkPatrick-Seidel:

Jarvis March:

Test Case 3:
